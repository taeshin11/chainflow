Date: 2026-04-19
Scope: Bloomberg-grade real-time institutional flow stack (bypass 13F's 45-day lag)

─────────────────────────────────────────────────────────────
MOTIVATION
─────────────────────────────────────────────────────────────
User wants to replicate what Bloomberg does for "real-time 수급 추적".
Reality: Bloomberg CANNOT show real-time 13F — the 45-day SEC deadline is
legally binding. What Bloomberg DOES do is stitch together:
  - Form 4 (insider trades, D+2)
  - Schedule 13D/13G (5%+ ownership crossings, +10 days)
  - Options unusual flow (real-time)
  - Block trade tape (real-time)
  - Korean/non-US exchanges that publish investor-type data real-time

This commit builds that same stack using free EDGAR data + one optional
paid API, and wires it into a new /insider page + AI report.

src/lib/edgar-insider.ts
  - parseAtomFeed(xml) — parse EDGAR getcurrent Atom entries
  - fetchRecentForm4({ feedCount, includeOther, tickersOnly })
      → crawls Form 4 RSS, resolves ownership.xml per filing, regex-parses
        nonDerivativeTransaction blocks. Returns InsiderTransaction[] with
        direction (buy/sell), insider role, shares, price, USD value.
  - fetchRecentOwnershipAlerts({ minPercent, tickersOnly })
      → crawls SC 13D/G/A feeds, downloads primary text, regex-extracts
        filer, CUSIP, percent owned, shares owned. Default >=5% filter.
  - resolveTickerFromIssuerName(name) — maps issuer name → ticker using
    TICKER_TO_COMPANY from edgar-13f.ts
  - Uses existing EDGAR_UA (FlowVium/1.0 taeshinkim11@gmail.com) and stays
    under SEC's 10 req/s via paced fetch + 6-8 concurrency batches.

src/lib/unusual-whales.ts
  - fetchOptionsFlow(limit) — calls api.unusualwhales.com/api/option-trades/
    flow-alerts if UNUSUAL_WHALES_KEY env is set; returns [] otherwise so
    the rest of the site works unchanged. Derives sentiment from side+type
    (ask-call = bullish, bid-call = bearish, etc.)

─────────────────────────────────────────────────────────────
WHY EACH SOURCE
─────────────────────────────────────────────────────────────
Form 4
  - Strongest single real-time signal: CEO/CFO open-market BUY (P code) of
    their own company. Historically beats market.
  - D+2 filing means we see the trade within 2 business days — 44 days faster
    than the quarterly 13F.
  - Filter: transactionValueUsd > $100K (noise cut).

Schedule 13D / 13G
  - Triggered when any entity crosses 5% ownership. Activist investors
    (Icahn, Ackman, Loeb) are legally required to file within 10 days.
  - 13D = active intent (board seat, M&A push); 13G = passive.
  - These filings often predate a news cycle by weeks.

Options Unusual Flow
  - Biggest unusual options sweeps are the institutional tell. If a $5M+
    call sweep hits NVDA the day before earnings, someone knows something.
  - Only reliable paid API for retail is Unusual Whales ($48/mo).
  - We left the integration in "awaiting key" state — zero cost until
    UNUSUAL_WHALES_KEY is set.

Korea Flow
  - KRX publishes foreign vs institutional vs retail net-buy amounts
    **intraday with only 15-min lag**. Unlike the US, there is no regulatory
    delay for this aggregate data.
  - 외국인 순매수 상위 종목은 한국 투자자에게 가장 중요한 실시간 신호.
