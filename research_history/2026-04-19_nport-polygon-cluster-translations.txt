NEW FILES
─────────────────────────────────────────────────────────────
src/lib/edgar-nport.ts
  - fetchRecentNPORT({feedCount}) — walks NPORT-P RSS, downloads primary_doc.xml
    per filing, regex-parses invstOrSec blocks, filters to our CUSIP map,
    returns NPortFundSnapshot[] per fund with {seriesName, regName, repPdDate,
    totalNetAssets, holdings[]}.
  - aggregateByTicker(snapshots) → NPortTickerAggregate[] showing "for each of
    our tickers, which mutual funds hold it with what value and % NAV".
  - Handles large XMLs (100-500KB per fund) in batches of 4 concurrent.

src/lib/polygon.ts
  - polygonKey() env gate — POLYGON_KEY; returns [] unless set.
  - fetchBlockTrades(ticker, minShares=10000, maxPages=2) — uses v3/trades
    endpoint, filters to >=10K share prints.
  - fetchBlockTradesForTickers(tickers) — parallel across tracked tickers.

src/app/api/nport-holdings/route.ts
  - GET returns {funds, byTicker, updatedAt}. Redis 6h TTL.

src/app/api/block-trades/route.ts
  - GET returns {items, configured, total}. {configured:false} when no key.
  - Redis 5min TTL. Tracked roster of 27 semi/AI/defense/bio/energy tickers.

scripts/fix-locked-msg.py
  - Rewrote optionsLockedTitle/Body to funding-goal prose in all 16 languages.

scripts/add-insider-v2-i18n.py
  - Added tabNport / tabBlocks / blocksLockedTitle / blocksLockedBody /
